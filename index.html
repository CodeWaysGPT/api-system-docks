Hello Generator API Documentation

Get a Random Hello Message

To get a random hello message, make a GET request to the following URL:

GET /hello/api/:key

Replace :key with a valid API key.

Response:

{
  "Hello": "Hello, World!"
}

Generate a Random API Key

To generate a random API key, make a GET request to the following URL:

GET /hello/genkey

Response:

{
  "apiKey": "your_generated_api_key"
}

Notes:

  • The API key is required for authentication when accessing the random hello message route.
  • The generated API key should be securely stored and used to authenticate API requests.
  • The API key is unique and should be kept confidential to prevent unauthorized access the API.

Example Usage:

Note: Replace "your-api-key" in the example usage section with the actual API key you've generated or obtained from the /hello/genkey route.

// Get a random hello message
{"Hello":"9. Good afternoon"}
